Turn agent-run evidence into a deterministic pass or fail.

Agent Eval Kit is a standalone Python application that validates a versioned task contract, seals it with SHA-256, and evaluates recorded evidence against the pinned repository commit, exact command, and required checks. It runs offline and has no third-party runtime dependencies.

The caller runs tests in its own isolated environment; Agent Eval Kit validates only recorded JSON evidence and never executes customer code.
